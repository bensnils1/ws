About This Website

Credit is due to MathJax, along with the polyfill given in their docs for older browser support, for any maths typesetting here (eg. \(\LaTeX\)). I've also taken advantage of Tailwind to get everything into a (hopefully) presentable condition for most devices with as little headache as possible, and the fonts used across the site include Lora and DM Serif Display, both available through Google Fonts. Finally, the website is hosted from a Github repository with Pages.
